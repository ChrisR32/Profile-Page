A little program I made as I wondered how Random Ruby-Random is.
- Saves and loads the result.
- Displays Random error messages as a way to display who was picked.
- Displays different emojiis depending on total hits.
- Navigation controlled by the Up and Down Keys, and selection made with the Enter Key.

Visit my Git Hub page to learn more.

Originally created for my Terminal Application project at Coder Academy. I find that I'm always playing around with this, always trying to figure out that next problem, or creating them which is normally the case. Lately I've been working on fully custom a menu system to get rid of the number based system that the first program used, I was never a fan of the Number System and only used it due to time constraints.
The concept remains the same:
- A tool that lets you SSH into your server and view stats.
- A dashboard screen that shows you the normal every day type info.
- Backs up the scans in logs for checking later

Anyhow, Checkout the videos to compare the menus, let me know what you think.
